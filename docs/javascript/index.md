# JavaScript 和 TypeScript 教程

欢迎来到 JavaScript 和 TypeScript 教学文档中心！这里提供了从基础到进阶的 JavaScript 和 TypeScript 学习资源，帮助您掌握这两种重要的编程语言。

## 文档分类

- [JavaScript 基础](./js-basics) - JavaScript 基本语法、数据类型和核心概念
- [JavaScript 进阶](./js-advanced) - 闭包、原型链、异步编程等高级主题
- [ES6+ 新特性](./es6-plus) - ES6 及以上版本的新特性和语法糖
- [DOM 操作](./dom) - 使用 JavaScript 操作 DOM 元素和事件处理
- [TypeScript 埀门](./ts-basics) - TypeScript 基础类型、接口和类
- [TypeScript 进阶](./ts-advanced) - 泛型、装饰器、模块系统等高级特性
- [最佳实践](./best-practices) - JavaScript 和 TypeScript 编码规范与优化技巧

## 学习路径建议

1. **初学者**：从 [JavaScript 基础](./js-basics) 开始，了解 JavaScript 的基本概念和语法
2. **有一定经验的开发者**：学习 [ES6+ 新特性](./es6-plus) 和 [DOM 操作](./dom)
3. **准备学习 TypeScript**：先掌握 JavaScript 基础，然后学习 [TypeScript 埀门](./ts-basics)
4. **专业开发者**：深入学习 [JavaScript 进阶](./js-advanced) 和 [TypeScript 进阶](./ts-advanced)，并阅读 [最佳实践](./best-practices)
