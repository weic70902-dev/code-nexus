# HTML 教学文档

欢迎来到 HTML 教学文档中心！这里提供了从基础到进阶的 HTML 学习资源，帮助您掌握网页标记语言的各项技能。

## 文档分类

- [HTML 基础](./basics) - HTML 基本语法、标签和结构
- [HTML 表单](./forms) - 表单元素和验证
- [HTML5 新特性](./html5) - HTML5 语义化标签、多媒体元素等
- [最佳实践](./best-practices) - HTML 编写规范和优化技巧

## 学习路径建议

1. **初学者**：从 [HTML 基础](./basics) 开始，了解 HTML 的基本概念和语法
2. **进阶用户**：学习 [HTML5 新特性](./html5) 和 [表单处理](./forms)
3. **专业开发者**：阅读 [最佳实践](./best-practices) 来提升代码质量
