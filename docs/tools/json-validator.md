# JSON 验证工具

JSON 验证工具可以帮助您验证 JSON 数据的有效性，确保数据格式正确无误。

## 功能特性

- JSON 语法验证
- 错误定位和提示
- 行号指示
- 实时验证

## 使用方法

1. 在输入框中粘贴您的 JSON 数据
2. 工具会自动检测数据有效性
3. 如果数据有效，会显示成功消息
4. 如果数据无效，会显示错误信息和位置

## 错误类型

- **语法错误** - 如缺少括号、逗号等
- **格式错误** - 如键名未用双引号包围
- **类型错误** - 如使用了不支持的数据类型

## 示例

### 有效的 JSON
```json
{
  "name": "John",
  "age": 30,
  "city": "New York"
}
```

### 无效的 JSON
```json
{
  "name": "John",
  "age": 30,
  "city": "New York",
  // 缺少闭合大括号
```

## 注意事项

- 仅支持标准 JSON 格式
- 不支持注释（JSON 标准不支持注释）
